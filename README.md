# GoGoDiary

- GoGoDiary致力於解決記錄的問題，搭配地圖資訊、標籤功能，幫您記住生活的點點滴滴。

- 地圖探索：提供巷弄間的美食資訊、周遭景點

- 口袋清單：記錄下次要拜訪的地方，讓你不健忘

- 草稿清單：尚未編輯完成的日記，一目瞭然

- 標籤系統：幫日記做分類，找尋記憶更方便
